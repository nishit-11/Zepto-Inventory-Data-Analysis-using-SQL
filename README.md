# 🛒 Zepto Sales Analysis using PostgreSQL
This project focuses on analyzing a retail product dataset from Zepto using PostgreSQL. The dataset was imported from a CSV file into a PostgreSQL database, followed by SQL-based exploratory data analysis to uncover meaningful business insights related to pricing, discounts, inventory, and product categories.

📌 Objectives
Import and manage retail data in PostgreSQL.
Perform data exploration and validation.
Analyze product pricing, discounts, and inventory.
Generate category-wise business insights using SQL.
Practice writing optimized SQL queries for analytical tasks.

🛠️ Tech Stack
- PostgreSQL
- SQL
- CSV Dataset
- pgAdmin 4

📊 Key Analysis Performed
- Total number of products
- Category-wise product distribution
- Revenue analysis by category
- Highest and lowest priced products
- Average discount by category
- Out-of-stock product analysis
- Inventory availability analysis
- Price and discount comparisons
- Aggregate analysis using SUM(), AVG(), COUNT(), MAX(), and MIN()
- Data filtering, sorting, grouping, and conditional queries
